# Getting Started with LangChain

:::info Docs coming soon
The OpenBox SDK for [LangChain](https://www.langchain.com/) is in development.
This page will be updated with a full getting-started guide when the integration is available.
:::

OpenBox will integrate with LangChain by wrapping chain execution — your agents, tools, and prompts stay exactly as they are while every step is governed, scored, and auditable.

## What to expect

- Governance layer for LangChain chains and retrieval pipelines
- Trust scoring and policy enforcement for every agent action
- Full session replay across chain steps
- HTTP call recording for all LLM and tool invocations
