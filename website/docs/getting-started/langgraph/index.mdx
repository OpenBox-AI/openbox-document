# Getting Started with LangGraph

:::info Docs coming soon
The OpenBox SDK for [LangGraph](https://www.langchain.com/langgraph) is in development.
This page will be updated with a full getting-started guide when the integration is available.
:::

OpenBox will integrate with LangGraph by wrapping graph execution — your agents, nodes, and state machines stay exactly as they are while every step is governed, scored, and auditable.

## What to expect

- Governance layer for LangGraph state machines and agent graphs
- Trust scoring and policy enforcement for every node execution
- Full session replay across graph traversals
- HTTP call recording for all LLM and tool invocations
