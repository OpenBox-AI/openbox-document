# Getting Started with Cursor

:::info
The Cursor integration is open source.
Refer to the README for full setup instructions:
**[OpenBox-AI/cursor-hooks](https://github.com/OpenBox-AI/cursor-hooks)**
:::

OpenBox integrates with Cursor via the official [hooks system](https://docs.cursor.com/context/hooks) — your IDE workflows stay exactly as they are while every agent action is governed, scored, and auditable.

## What to expect

- Install with a single command — no code changes to your Cursor setup
- Every prompt, file read, shell command, and MCP tool call is governed before execution
- PII/secret redaction on file content and MCP responses
- Human-in-the-loop approvals for sensitive operations
- Goal alignment scoring and drift detection
- Full session replay and audit trail
